1.XML - расширяемый язык разметки. Используется в SOAP и REST-запросах. JSON - текстовый формат обмена данными, основанный на JavaScript. Также используется в REST-запросах. JSON по сути является универсальным словарём, то есть чтобы переформатировать данные из одного языка программирования в другой, сначала форматируют первый в JSON, а затем уже из JSON во второй язык программирования.
2.filesystem - обширная библиотека, позволяющая копировать/удалять файлы, создавать новые файлы и директории, показывать абсолютный и относительный пути к файлу, работать с итератором по файлу, выводить название и расширение файлов, размер файлов, применять различные булевские операции и прочее.
